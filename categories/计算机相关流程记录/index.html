surface go2 安装macOS

这是第一次安装系统，中间可能有问题，目前仅限于安装好进入系统。

  • 相关设置及详细说明主要参考 https://github.com/jalower/Surface_Go_2_Opencore

  • surface go2 (奔腾Y4425处理器) 上的EFI 文件 https://github.com/jlempen/Surface-Go-2-OpenCore

  • 安装流程参考小红书帖子 https://www.xiaohongshu.com/discovery/item/671dd191000000001b011efc

  • 由于网上很少有在奔腾处理器上安装macOS的经验，所以有一些问题。经尝试，macOS 12可以成功安装，安装macOS 13或者从12升级到13都会失败，表现为安装到最后会反复重重启，最后失败报错。

待解决的问题

  • 触屏失效
  • 有时候会非常卡顿，可能由于温控导致降频，可尝试关掉温控
  • 界面显示的字体太小，但系统设置-显示-缩放，没有别的分辨率可选（屏幕还是系统的问题？）
  • 尝试其他 macOS版本

目前可以使用

  • wifi
  • 蓝牙
  • 登录apple id，同步

详细步骤

准备工作

  1. 制作macOS安装u盘：

    1. 准备32G u盘，可以直接用mac自带软件磁盘工具Disk Utility （抹掉 Erase），填u盘名称，格式为mac OS扩展（区分大小写，日志式）

    2. 直接在App store下载 macOS各版本的系统

    3. 制作u盘，直接运行不同版本macOS对应的命令，参考苹果官网：https://support.apple.com/zh-cn/101578 ，例如macOS 12 Monterey 对应命令如下，其中MyVolume为前面自己设置的u盘名称

  2. 制作PE系统u盘（在Windows上电脑有很多软件可以制作，但这些软件在mac上不能用，所以如何在mac电脑上制作？）

    1. 用到的工具 balenaEtcher：https://etcher.balena.io/
    2. 下载pe安装包（这个是别人制作好的的） [wepe for mac.dmg](../../../Desktop/wepe for mac.dmg)
    3. 在balenaEtcher 选择安装包，选择u盘，直接flash即可

  3. EFI 文件

    1. 主要在前面两个 github 上下载，对应处理器的
    2. 其中序列号等需要自己生成，参考 https://dortania.github.io/OpenCore-Install-Guide/config.plist/#creating-your-config-plist
    3. 用到的生成工具 GenSMBIOS, 安装后双击运行GenSMBIOS.command，选择Generate SMBIOS，生成相应内容
    4. 编辑工具 ProperTree , (但我不太会用，直接复制粘贴过去的)

开始安装

  1. 开机后长按 音量+ 键，到UEFIs设置，必须设置 Secure Boot disabled，Boot configuration 将 USB Storage移到最上
  2. 插上PE u盘，Exit 重启，选择PE系统安装
  3. 安装好之后，在PE系统，通过DiskGenius软件进行磁盘分区。快速分区，分区数目1个，勾选创建新ESP分区（这个有90多M，看是否能放下EFI文件）
  4. 打开我的电脑，找到这个ESP分区，将EFI 文件文件拷进去
  5. 插上macOS 系统u盘，重启，选择macOS 安装，选择 磁盘工具Disk Utility，erase抹掉 磁盘，重命名，设置格式为APFS
  6. 左上角退出磁盘工具，回到前一页面，选择安装macOS，选择要安装的位置
  7. 开始安装，这中间可能会有几次反复重启，理论上不用管，最后安装完就是进入系统，选择语言等设置。
